# 🛒 Customer Shopping Analysis using Python

## 📌 Project Overview

This project analyzes customer shopping data using Python, Pandas, and Matplotlib. It includes data cleaning, sales analysis, and data visualization to understand customer purchasing behavior and generate business insights.

---

## 🎯 Objectives

- Clean and preprocess customer shopping data
- Calculate total revenue and average purchase value
- Analyze sales by category, city, and gender
- Identify top-selling products
- Analyze payment method usage
- Visualize business insights using charts

---

## 🛠️ Technologies Used

- Python
- Pandas
- Matplotlib

---

## 📂 Dataset

The dataset contains the following columns:

- Customer_ID

## 🚀 How to Run the Project

1. Clone the repository

```bash
git clone https://github.com/NMSana/Customer-Shopping-Analysis.git
```

2. Install the required libraries

```bash
pip install -r requirements.txt
```

3. Run the project

```bash
python customer_shopping_analysis.py
```
